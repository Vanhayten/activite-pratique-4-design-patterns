# Activité Pratique N°4 : Design Patterns & AOP

Ce dépôt contient le travail de réalisation de l'activité pratique N°4 et de l'examen blanc du module Design Patterns de la filière d'ingénieur GLSID (Pr. Mohamed YOUSSFI).

## Contenu

1. **Exercice 1** : Modélisation UML des Design Patterns pour des situations spécifiques.
2. **Exercice 2** : Framework de traitement d'images (Strategy, Template Method, Adapter).
3. **Examen Blanc** : Application de manipulation de dessins.
   - Composite, Observer, Strategy.
   - Sérialisation.
   - Programmation Orientée Aspect (AOP) pour la journalisation et la sécurité.
   - Interface Graphique avec JavaFX.

## Structure du projet

Le projet est divisé en sous-modules Maven pour chaque exercice.
